# Orders-and-sales-data-Analysis

This project analyzes Store sales data using **SQL, Excel, and Power BI** to identify business insights such as sales trends, customer behavior, top-performing channels, and order success rate.

The dataset contains order details like customer info, order status, category, amount, shipping location, sales channel, and more.

# Tools Used
- MySQL
- Microsoft Excel
- Power BI


 Project Files
- SQL Queries.txt
- Excel Dashboard.xlsx
- Power BI Dashboard.pbix
- Dataset.csv

- Success Rate

****Power BI Dashboard
- Revenue KPI
- Orders KPI
- Cancel Rate
- Average Order Value
- Success Rate
- Sales by Channel
- Sales by State
- Monthly Revenue Trend
- Gender Analysis

**** Key Insights
- Women customers placed more orders than men.
- Myntra and Amazon generated highest revenue.
- Delivered orders were significantly higher than cancelled orders.
- Maharashtra and Karnatka contributed major share in sales.
- Kurta and Set performed better than others.
